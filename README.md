# Catálogo web Patitas Ya

Incluye:

- Diseño adaptable a celular y computador.
- Búsqueda de productos.
- Filtros por categoría.
- Ordenamiento por precio o nombre.
- Carrito persistente con localStorage.
- Pedido directo por WhatsApp.
- Formato de precios en pesos chilenos.

## Cómo usarlo

1. Abre `index.html` en tu navegador.
2. Edita los productos dentro de `app.js`.
3. Reemplaza `56900000000` por el número real de WhatsApp.
4. Cambia colores, textos y estilos dentro de `styles.css`.

Para publicar gratis puedes usar GitHub Pages, Netlify o Vercel.
